Objective

We were challenged with pseudocoding a task we would like to accomplish on our computer through Java Script.

Problems

With this lab we did not encounter any challenges. While we are both busy outside of school, we were able to find enough time to work on the lab and complete it successfully.

Results

We Decided that we wanted to code a screen where a logo bounces from border to border infinitely like the DVD pause screen. This is what our psuedocode looks like.

Our Pseudocoded Everyday Task

    How to use a 3D Printer

  1. Find a file you would like to print
  2. Download the file onto your computer
  3. Upload the file into Cura or another slicer program
  4. Adjust the settings based on personal preferences
  5. Download the file on to an SD card
  6. Insert the SD card into the printer
  7. Turn the printer on
  8. Level the print bed
  9. Disable steppers
  10. Lay a piece of paper on print bed
  11. Adjust the bed so the nozzle barely touches the paper
  12. Select the file on your printer
  13. Wait for the print to finish.

Our Pseudocoded Computer Task

    Making a Logo bounce off of border

  1. Load the logo on the screen
  2. Move the logo in a random direction
  3. Check if the logo is hitting a wall
  4. If in contact with a wall, bounce off the wall
  5. If not, continue moving
  6. Loop FOREVER
